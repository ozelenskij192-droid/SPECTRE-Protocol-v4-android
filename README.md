# SPECTRE: Protocol v4.1 - Tactical Mil-Sim Prototype

Ultra-realistic tactical shooter inspired by Ready or Not, Squad, and Arma. Built in Godot 4.3 for PC & Android.

## Features
- Super-realistic ballistics, wounds, AI
- Co-op & multiplayer modes
- VR support planned
- Real weapon models from Blender
- Android touch controls (virtual joystick + buttons)

## Setup
1. Download Godot 4.3: https://godotengine.org/
2. Open project.godot
3. Install Virtual Joystick addon from AssetLib (for mobile)
4. Run world.tscn (F5)

## Android Export
- Editor Settings → Export → Android (setup JDK & SDK)
- Project → Export → Android → spectre.apk

## Models
- M4A1: Download .glb from Sketchfab and place in models/weapons/
- Soldier: Download .glb from Sketchfab and place in models/characters/

Full game concept in docs/game_concept.md
Python prototypes in prototypes/ for reference.
